## 🎯 What is FreeKiosk?

FreeKiosk is an open-source kiosk platform for Android tablets, built for teams that want **full control without license costs**. It's the free alternative to Fully Kiosk Browser with enhanced automation capabilities.

### Why teams choose FreeKiosk

- ✅ **Free and MIT licensed** - No per-device costs or licensing fees
- ✅ **Device Owner support** - True lockdown capabilities for enterprise deployments  
- ✅ **Multiple kiosk modes** - WebView, External App, Dashboard, and Media/Multi-App modes
- ✅ **Rich automation** - 40+ REST endpoints and MQTT integration with Home Assistant discovery
- ✅ **Headless provisioning** - ADB-based configuration for mass rollout
- ✅ **Privacy-first** - No mandatory tracking or cloud dependencies

### FreeKiosk vs Fully Kiosk Browser

| Feature | FreeKiosk | Fully Kiosk |
|---------|-----------|-------------|
| Price | 🟢 Free | 🔴 Paid per device |
| Open-source | 🟢 MIT | 🔴 Closed source |
| Device Owner mode | ✅ | ✅ |
| REST API | ✅ | ✅ |
| MQTT + Home Assistant discovery | ✅ | ❌ |
| Cloud fleet management | Roadmap | ✅ |

### Tech Stack

- **React Native + TypeScript** - Cross-platform mobile framework
- **Kotlin native modules** - Android policy and system control
- **Android SDK 26+** - Compatible with Android 8.0+
